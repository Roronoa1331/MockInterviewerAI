# 前端核心技术栈说明

本岗位的核心技术栈是根据实际项目开发与真实业务需求提炼得来的，主要涵盖以下几个维度：

## 1. 深入的 JavaScript 与 Web 基础
- **JavaScript 底层原理**：深入理解执行上下文、执行栈、作用域链、闭包机制及其垃圾回收（V8 新老生代 GC 算法）。透彻理解 `this` 的五种绑定规则，并具备手写 `call/apply/bind`、`深拷贝` 等原生函数的能力。
- **ES6+ 进阶与异步编程**：熟练掌握 `let/const` 的块级作用域底层原理、`Proxy` 数据代理；精通异步编程模型，深刻理解事件轮询机制（Event Loop 的宏任务与微任务），并能手写实现 `Promise`（含链式调用、`all/race`、`retry`）及理解 `async/await` 的 Generator 语法糖原理。
- **浏览器进阶与模式**：掌握 Web Worker 以解决主线程高计算阻塞问题；理解 JS 沙箱（Sandbox）原理及其使用场景；熟悉 Hybrid App 中的 JSBridge 双向通信底层原理。
- **CSS 架构与布局**：熟练运用 BEM 命名规范维护中大型项目样式；掌握 `rem+vw` 响应式布局方案；深入理解 CSS3 硬件加速（GPU加速）、BFC 机制、Sticky 粘性布局、以及如何通过样式规避回流（Reflow）和重绘（Repaint）。

## 2. 主流前端框架及底层架构
- **Vue.js 深入剖析**：
  - **响应式原理**：精通 Vue2 `Object.defineProperty` 与 Vue3 `Proxy` 的数据劫持机制，能手写 mini 版 MVVM 框架和 `v-model` 双向绑定。
  - **更新机制**：深入理解 Vue 的异步更新队列原理及 `$nextTick` 的降级策略（Promise -> MutationObserver -> setImmediate -> setTimeout）。
  - **模块原理**：熟练掌握虚拟 DOM 和 Diff 算法底层逻辑；深入理解 `computed` 的缓存控制（dirty标记）与 `watch` 原理；掌握 CSS Scoped 的 AST 转化机制。
  - **生态系统**：了解 Vuex 的状态管理注入原理及 Vue-Router（Hash/History）的路由机制。
- **React.js 核心机制**：
  - **Hooks 与函数式编程**：理解 React 推崇函数式组件的原因，熟练掌握 `useMemo`、`useCallback` 等 Hooks 进行渲染性能优化。
  - **Fiber 架构**：深入理解 React Fiber 采用大任务拆分、基于 `requestIdleCallback` 机制来解决旧版更新掉帧的问题。
  - **状态与编译**：理解 `setState` 的同步/异步表现（合成事件 vs 原生事件），以及 JSX 的编译转换本质。

## 3. 前端工程化与构建体系
- **Webpack/Vite 构建原理**：
  - 掌握 Webpack 的完整打包流程（从 AST 解析到依赖图谱生成）；具备手写 `Loader`（源码转换）与 `Plugin`（基于 Tapable 的事件流注册）的能力。
  - 理解 Vite 基于 ES Module 的无包构建原理和毫秒级热更新（HMR）优势。
  - 了解 Webpack5 的模块联邦（Module Federation）特性在微前端中的应用。
- **Babel 编译机制**：理解 AST 抽象语法树的作用，掌握 Babel 解析、转换、生成的三阶段工作流，了解如何编写自定义 Babel 插件。
- **工程化工具与自动化**：掌握 Gulp 在组件库构建（如一键换肤合并样式）中的应用，了解基于 Commander、Inquirer 实现自定义团队脚手架的开发流程。

## 4. 性能分析、优化与监控
- **性能评估与指标**：熟练使用 Lighthouse 和 Chrome Performance 工具排查性能瓶颈。掌握 Web Vitals 核心指标：FP、FCP、LCP、TTI、CLS 的计算及其业务意义。
- **加载与渲染优化**：掌握资源预加载（`preload` / `prefetch`）、脚本异步加载（`async` / `defer` 的执行时机差异）、路由按需懒加载、HTTP 缓存策略（强缓存与协商缓存如 Etag/Last-Modified 优先级）。
- **前端监控体系**：了解白屏时间与首屏时间的算法计算（基于 MutationObserver），掌握 requestIdleCallback 的应用以及图片的动态打点上报机制。

## 5. 计算机网络与网络安全
- **网络协议底座**：透彻理解 TCP 的三次握手、四次挥手及 2MSL 等待时间；理解 UDP 与 TCP 的区别；掌握 HTTP/1.x（Keep-Alive）、HTTP/2（多路复用、头部压缩）、HTTP/3（QUIC）的演进历程。了解 DNS 解析与 HTTPS 的非对称/对称加密握手流程。
- **安全防范机制**：
  - 掌握跨域解决方案（CORS 简单/非简单请求及预检）。
  - 精通 XSS（跨站脚本攻击）与 CSRF（跨站请求伪造）的攻击原理及防御手段（转义、白名单、Token、Cookie SameSite 配置）。
  - 了解 CSP 内容安全策略以及中间人攻击的原理。

## 6. Node.js 全栈与服务端渲染
- **服务端渲染 (SSR)**：了解 CSR、SSR、SSG、ISR 等渲染模式的优缺点及应用场景。熟悉 Nuxt.js/Next.js 等框架同构渲染的生命周期特性及 SEO 优化。
- **Node.js 基础**：掌握 Node.js 的事件循环模型及高并发原理；理解 Express/Koa 的洋葱圈中间件模型；了解大文件断点续传（Pipe 管道流）、接口防刷等常见业务方案。了解数据库基础（MongoDB 与 MySQL 区别，Redis 的高频应用场景）。
