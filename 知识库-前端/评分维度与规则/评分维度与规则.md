# AI面试评分维度与规则

在AI面试评估过程中，系统将根据以下5个核心维度对候选人进行全面打分。每个维度具有不同的权重，最终的总分将由各个维度的得分乘以对应权重后累加得出。

## 一、 评分维度与权重分配

1. **技术知识储备 (technical_knowledge) - 35%**
   - **权重**：0.35
   - **说明**：评估候选人对该岗位核心技术栈、基础理论、框架原理的掌握程度及深度。

2. **问题解决能力 (problem_solving) - 25%**
   - **权重**：0.25
   - **说明**：评估候选人在面对具体业务场景、Bug排查或算法问题时，分析问题、拆解问题并给出有效解决方案的能力。

3. **沟通表达能力 (communication) - 20%**
   - **权重**：0.20
   - **说明**：评估候选人表述是否清晰、逻辑是否严密，能否准确理解面试官（AI）的问题并进行有效的互动。

4. **代码质量与规范 (code_quality) - 10%**
   - **权重**：0.10
   - **说明**：评估候选人编写代码（或口述代码思路）时的规范性、可读性、健壮性以及对工程化和设计模式的理解。

5. **综合能力评估 (overall_competence) - 10%**
   - **权重**：0.10
   - **说明**：结合候选人的项目经验、学习能力、潜力及与岗位的整体匹配度进行的全面评估。

## 二、 总分计算规则

系统采用加权求和（Weighted Sum）的方式计算候选人的最终面试得分（`overall_score`）。

**计算公式：**
```python
overall_score = sum(scores[aspect] * weights[aspect] for aspect in scores)
```

**解析：**
最终总分 = (技术知识得分 × 0.35) + (问题解决得分 × 0.25) + (沟通表达得分 × 0.20) + (代码质量得分 × 0.10) + (综合能力得分 × 0.10)
