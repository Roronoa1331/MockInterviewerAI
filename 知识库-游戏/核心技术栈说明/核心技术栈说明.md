# 游戏开发核心技术栈说明

本岗位的核心技术栈是根据实际项目开发与真实业务需求提炼得来的，主要涵盖以下几个维度：

## 1. 编程语言基础与内存管理
- **C# 语言进阶**：深入理解值类型与引用类型在内存布局与赋值语义上的区别；精通面向对象思想（多态的实现、虚函数、接口与抽象类）；熟练掌握泛型缓存、反射（以及在 IL2CPP 环境下的限制与注意点）；掌握 C# 委托、事件与 Lambda 表达式（及其闭包陷阱）；精通 LINQ 的合理使用，以及 `async/await` 异步编程与异常处理的规范。
- **内存与垃圾回收 (GC)**：透彻理解 C# 分代垃圾回收（Generational GC）机制与工作流程（标记-清除-压缩）；深入理解 Finalizer（析构函数）与 `IDisposable` 的资源管理模式；了解 Unity 的 Boehm GC 以及 Incremental GC 的区别；具备防范及处理内存泄漏、`MissingReferenceException` 和消除高频逻辑装箱 (Zero GC) 的能力。
- **C++ 语言核心**：掌握 C++ 面向对象模型，深入理解多态、动态分派（Dynamic Dispatch）、虚表（vtable）内存布局机制，以及析构过程中的异常安全性处理。
- **Lua 热更脚本**：熟练掌握 Lua 的表（Table）与元表（Metatable）机制以模拟面向对象开发；理解浅拷贝与深拷贝的本质；了解 Lua 5.x 垃圾回收周期原理；熟练应用 xLua 等热更方案，处理 C# 与 Lua 的互调性能与类型映射。

## 2. 主流游戏引擎（Unity & Godot）
- **生命周期与组件系统**：精通引擎（如 Unity MonoBehaviour）的生命周期（Awake, Start, Update, FixedUpdate 等），了解物理逻辑与画面更新的最佳插入点。深入理解组件式架构（组合优于继承）在实体角色设计中的应用。
- **协程与异步模型**：透彻理解协程（Coroutine）与操作系统线程的本质区别、实现原理及在资源异步加载（如 AssetBundle）中的应用，了解 UniTask 在游戏开发中的优势。
- **资源与UI管理**：
  - 掌握 AssetBundle/Addressables 的打包、压缩策略、依赖关系建立及安全的异步加载与卸载流程。
  - 理解 ScriptableObject 在配置数据与状态管理中的妙用。
  - 精通 UGUI 的渲染模式（Canvas Render Mode）、缩放适配（Canvas Scaler）、事件传递及 UI 合批优化手段。

## 3. 图形学、渲染与特效
- **渲染管线与着色器 (Shader)**：深入理解渲染管线从模型空间到齐次裁剪空间的坐标变换流程；明确顶点着色器（Vertex Shader）与片元着色器（Fragment Shader）的分工；熟知内置渲染管线、SRP（URP/HDRP）的选择与使用。
- **图形处理机制**：
  - 精通 Alpha Blend（透明混合）与 Alpha Test（裁剪透明）的原理区别及其适用场景。
  - 理解本影、半影的物理概念及其在阴影计算中的实现思路。
  - 了解 PBR（基于物理的渲染）、全局光照（GI）、发光特效（Bloom/Glow/HDR）、屏幕空间反射等高级特效。
- **性能优化**：深入理解 Draw Call（绘制调用）的产生原因及其过高的影响，熟练运用静态合批、动态合批、GPU Instancing 等降 Draw Call 方案；了解 Skinned Mesh（蒙皮网格骨骼动画）的计算开销与优化。

## 4. 游戏物理、动画与 AI
- **物理系统**：理解刚体（Rigidbody）、碰撞体（Collider）和触发器（Trigger）的运作机制；掌握射线检测（Raycast）碰撞的数学原理和应用细节；理解并解决高速小物体撞击大物体（CCD机制）的问题。
- **动画系统**：了解游戏动画分类（关键帧动画、骨骼动画、2D 精灵动画）；掌握 Animator 状态机、动画层（Animation Layer）、动画融合（CrossFade / Blend）与 Humanoid 动画重定向（Avatar）原理。
- **游戏人工智能 (AI)**：
  - 熟练运用有限状态机（FSM）与行为树（Behavior Tree）实现复杂的 NPC/怪物逻辑。
  - 掌握寻路与导航技术（A* 算法原理，Godot 中的 `NavigationAgent2D` 动态避障与路径规划）。
  - 了解基于视野感知（RayCast/区域检测）的避障与索敌机制，以及 PCG（程序化地图生成）的核心思路。

## 5. 游戏工程架构与生产流
- **架构设计模式**：熟练掌握单例模式（如 GameManager）、观察者/发布订阅模式（事件/信号总线机制）、状态模式，了解这些模式在构建高内聚、低耦合游戏系统中的应用。
- **数字内容与工业化管线**：了解 3D 建模、动画、音效、UI 等数字资产的生产与集成工作流。熟悉编辑器工具开发（场景编辑器、数据配置导入工具），了解 DevOps 在游戏开发中的 CI/CD、自动化打出包流程。
- **服务端基础与网络同步**：了解游戏服务端的底层架构（如微服务、网关服、游戏服划分）；了解网络游戏同步策略，包括帧同步（Lockstep）与状态同步的基础知识，以及 AOI（兴趣区域）在降低同步数据量上的原理。
